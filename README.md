## IPMDAT
Invasive Plant Management Decision Analysis Tool

Dependencies:
* Python
* Flask (For Main App - Not Neccessary)

Directions:
* (Terminal) Proceed to project directory
* (Terminal) >>> python app.py
* (Browser)  localhost:8080/

OR

* (Terminal) Proceed to project directory
* cd SimpleHTTP
* python -m SimpleHTTPServer 8080
* (Browser)  localhost:8080/
